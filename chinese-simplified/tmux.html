1. tmux 是什么？

tmux 是一个 终端复用器：在一个终端窗口里同时运行多个互相独立的 shell，并排排布、可切换标签，并且（最关键的）与你的 SSH 连接解耦。

三大杀手锏：

  • 持久化 —— 合上笔记本，第二天重连，shell 仍停在你离开的地方
  • 布局 —— 把窗口切成多块面板；同时跑 vim、tail -f 和 htop
  • 可脚本化 —— 一条命令拉起整套项目布局、与同事共享会话、自动化环境搭建

2. 心智模型

tmux 是 客户端 / 服务端 架构。你的终端连接到一个常驻的 tmux 服务端进程。服务端持有：

服务端
└── 会话 "work"
    ├── 窗口 0：editor
    │   ├── 面板 0：vim
    │   └── 面板 1：shell
    └── 窗口 1：logs
        └── 面板 0：tail -f app.log

attach（附着） = 你的终端连入某个会话。detach（脱离） = 你的终端断开，但会话仍在服务端运行。关掉 SSH ≠ 杀掉会话。

3. 前缀键（prefix）

会话内几乎所有 tmux 命令都以 前缀键 开头。默认是 Ctrl+b。先按下前缀，松开，然后再按下一个键。

记号说明：C-b " 表示先按 Ctrl+b，再按 "（即 Shift+'）。许多人会把前缀重映射为 Ctrl+a，因为更顺手。

窗口类似浏览器标签页。每个会话可以有多个窗口，但同时只能激活一个。

6. 面板（panes）

面板把一个窗口切成多个矩形区域。tmux 的记号方式以「反直觉」著称：

  • C-b " 进行 水平分割（上下两块 —— 水平分割线）。记忆口诀：" 像两个上下叠的点。
  • C-b % 进行 垂直分割（左右两块 —— 垂直分割线）。记忆口诀：% 像两个方块中间一道斜杠。
动作键位
水平分割（上下）C-b "
垂直分割（左右）C-b %
在面板间移动C-b ↑/↓/←/→
循环到下一个面板C-b o
调整当前面板大小（按住）C-b C-↑/↓/←/→
切换缩放（面板全屏）C-b z
交换面板C-b { / C-b }
杀掉当前面板C-b x
把面板拆出为独立窗口C-b !
显示面板编号C-b q

7. 复制模式与回滚

鼠标固然可以滚动终端，但 tmux 的 正道 是 复制模式。用 C-b [ 进入。在里面可以用 vi 风格的键移动、查找、抓取文本。

动作键位（复制模式内）
进入复制模式C-b [
移动h j k l 或方向键
向前 / 向后查找/ / ?
开始选区Space（默认）或在 mode-keys vi 下用 v
复制选区Enter（默认）或 vi 模式下的 y
粘贴C-b ]
退出复制模式q

在 ~/.tmux.conf 中加上 setw -g mode-keys vi 启用 vi 风格。

8. 配置 — ~/.tmux.conf

在 tmux 内重新加载配置：按 C-b : 然后输入 source-file ~/.tmux.conf，或者绑定为快捷键。

# 把前缀重映射为 Ctrl-a
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# 窗口/面板编号从 1 开始（更靠近 1..9 数字键）
set -g base-index 1
setw -g pane-base-index 1
set -g renumber-windows on

# vim 风格的面板导航
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# 顺手的分屏键，并保留当前路径
bind | split-window -h -c "#{pane_current_path}"
bind - split-window -v -c "#{pane_current_path}"

# 鼠标 + vi 复制模式
set -g mouse on
setw -g mode-keys vi
bind r source-file ~/.tmux.conf \; display "config reloaded"

# 更大的回滚缓冲
set -g history-limit 50000

9. 插件（TPM）

TPM = Tmux Plugin Manager。在 ~/.tmux.conf 末尾追加：

set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'tmux-plugins/tmux-resurrect'   # 保存 / 恢复会话
set -g @plugin 'tmux-plugins/tmux-continuum'   # 开机自动恢复
run '~/.tmux/plugins/tpm/tpm'

tmux 内：C-b I 安装，C-b U 更新。tmux-resurrect 可以为整棵会话树打快照；tmux-continuum 每 15 分钟自动保存并在开机时重载 —— 笔记本重启 = 工作不丢。

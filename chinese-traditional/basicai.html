歡迎 — 用 PyTorch 寫下你的第一個神經網路

讀完這頁時,你將已經親手敲過一個完整神經網路的所有程式碼: 定義它、訓練它、執行推論。不准複製貼上 — 每個按鍵都重要。 在每個空格填上正確的 token,頁面會幫你計分,還有 連勝計數、動畫圖解,以及一個會在瀏覽器中「訓練」的最終小專案。

什麼是張量?
張量就是多維陣列 — 純量、向量、矩陣或更高維。 神經網路裡的一切都是張量:輸入、權重、梯度、輸出。 PyTorch 的 torch.Tensor 跟 NumPy 的 ndarray 很像, 但它能跑在 GPU 上,還會記住自己是怎麼產生的(autograd)。
什麼是模型?
一個帶有可學習參數的函數。輸入流經各層(矩陣相乘 + 非線性),產生輸出。我們透過比較輸出與真值、計算梯度、 往下坡推一點點權重的方式來調整參數。
訓練
餵給模型大量「輸入/目標」配對。對每一對:前向傳遞 → 計算損失 → 反向傳遞(梯度)→ 最佳化器步進(更新權重)。重複數千次。 損失曲線往下掉,就這樣。這就是深度學習。
推論
訓練完後,凍結權重、只對新輸入跑前向傳遞。更快(不用記梯度) 而且結果穩定。請使用 model.eval() 以及 torch.no_grad()。

建立模型 — 你的第一個 nn.Module

每個 PyTorch 模型都是 nn.Module 的子類別。 在 __init__ 裡定義各層,在 forward 裡定義前向傳遞。整個 API 就這樣。

10 類分類用的 2 層 MLP(完整)
import torch
import torch.nn as nn

class Net(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1  = nn.Linear(784, 128)
        self.relu = nn.ReLU()
        self.fc2  = nn.Linear(128, 10)

    def forward(self, x):
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)
        return x

model = Net()

損失與最佳化器

損失是個數字,告訴你「這個預測錯得多離譜」 — 越小越好。 最佳化器用梯度推一下權重,讓損失下降。 兩種損失涵蓋 90% 場景;兩種最佳化器涵蓋 95% 場景。

損失
迴歸 nn.MSELoss — 均方誤差
分類 nn.CrossEntropyLoss — 內建 softmax + log + nll
二元 nn.BCEWithLogitsLoss
最佳化器
簡單 optim.SGD(params, lr=0.01)
預設首選 optim.Adam(params, lr=1e-3)
現代 optim.AdamW — Adam + 解耦的 weight decay
組合起來
import torch.nn as nn
import torch.optim as optim

model     = Net()
loss_fn   = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-3)

model.train()

        pred = model(x)              # 1. 前向

把這五行敲出來

15 第 1 步 — 前向傳遞。把輸入餵進模型:
pred = (x)
16 第 2 步 — 計算預測與目標之間的損失:
loss = (pred, y)
17 第 3 步 — 清掉上一個 batch 的梯度(否則會累加!):
optimizer.()
18 第 4 步 — 反向傳遞,計算每個參數的新梯度:
loss.()
19 第 5 步 — 套用梯度,更新權重:
optimizer.()
常見 bug:如果忘了 optimizer.zero_grad(), 梯度會跨 batch 累加,模型會像在訓練「所有先前 batch 之和」一樣的東西。 永遠記得歸零。

推論 — 使用訓練完的模型

訓練結束後,你會想對新輸入跑模型,但不要計算梯度 (更快、更省記憶體),也不要讓 dropout / batchnorm 亂搞。兩個開關搞定。

推論模板
model.eval()                          # dropout 關閉,BN 改用 running stats
with torch.no_grad():                 # 不追蹤梯度
    logits = model(x_new)
    probs  = torch.softmax(logits, dim=-1)
    pred   = probs.argmax(dim=-1)        # 類別 id

小專案 — 用 XOR 問題訓練模型

XOR 是經典的「線性不可分」問題 — 單一感知器解不出來, 但兩層 MLP 可以。把最後幾個空格填好,完成這支程式。 當所有空格都變綠,頁面就會模擬訓練、即時顯示損失曲線下降, 以及模型如何學到決策邊界。

import torch
import torch.nn as nn

# 4 個點的 XOR 資料集
X = torch.tensor([[0.,0.],[0.,1.],[1.,0.],[1.,1.]])
Y = torch.tensor([[0.],[1.],[1.],[0.]])

class XORNet(nn.):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(, 8)
        self.fc2 = nn.Linear(8, )

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        return torch.sigmoid(self.(x))

optimizer = torch.optim.(model.parameters(), lr=0.05)

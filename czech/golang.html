Ahoj, Go

Malý staticky typovaný jazyk s velkým runtime: gorutiny, garbage collector, rychlý kompilátor a velkolepá standardní knihovna.

Dnešní výchozí stav: Go 1.22+ je praktická spodní hranice. Máte generika (1.18), any jako alias pro interface{}, ranged-for nad celými čísly (for i := range 10) a strukturované logování přes log/slog.

Proměnné a typy

Go je staticky typované, ale díky inferenci typů a krátké deklaraci působí lehčeji než Java.

Deklarace
var name string           // nulová hodnota: ""
var age int = 42
var ok = true             // typ odvozen

// krátký zápis (jen uvnitř funkcí)
name := "jane"
age, ok := 42, true

// blok
var (
    host = "localhost"
    port = 8080
)
Konstanty
const Pi = 3.14159
const Name string = "go"

// iota: automaticky inkrementuje 0,1,2,...
const (
    Sunday = iota   // 0
    Monday              // 1
    Tuesday             // 2
)

// typovaná bitová maska přes iota
type Flag uint8
const (
    Read   Flag = 1 << iota  // 1
    Write                       // 2
    Exec                        // 4
)

// celá čísla

Nulové hodnoty
Žádné „neinicializované“. Každá proměnná dostane typovou nulu.

Konverze typů
Žádná implicitní konverze — ani mezi číselnými typy. Buďte explicitní.

Řízení toku

Jedna smyčka, jedno if, jedno switch — ale každé nese svou váhu.

if (s init)
if err := doWork(); err != nil {
    return err
}

// proměnná v rozsahu zmizí po if
if v, ok := m["key"]; ok {
    fmt.Println(v)
} else {
    fmt.Println("missing")
}
for — jediná smyčka v Go
// klasická tříklauzulová
for i := 0; i < 10; i++ { … }

// while-style
for n < 100 { n *= 2 }

// nekonečná
for { … }

// range
for i, v := range slice { … }
for k, v := range m { … }
for _, line := range lines { … }
for i := range 10 { … }   // Go 1.22+
switch (bez fallthrough ve výchozím)
switch os := runtime.GOOS; os {
case "darwin":
    fmt.Println("mac")
case "linux", "freebsd":
    fmt.Println("unix")
default:
    fmt.Println(os)
}

// switch nad výrazy / podmínkami
switch {
case n < 0:   return "neg"
case n == 0:  return "zero"
default:      return "pos"
}
type switch
switch v := x.(type) {
case int:
    fmt.Println("int", v*2)
case string:
    fmt.Println("str", len(v))
case nil:
    fmt.Println("nil")
default:
    fmt.Printf("%T\n", v)
}
Návěští, break, continue, goto
outer:
for _, row := range grid {
    for _, cell := range row {
        if cell == target {
            break outer
        }
    }
}
Past (před 1.22): ve starším Go se proměnná smyčky znovu používala napříč iteracemi — zachycení v closure nebo gorutině pak vázalo stále stejnou adresu. Go 1.22 to opravilo tak, že každá iterace má čerstvé v. Pokud cílíte na starší Go, udělejte v := v uvnitř smyčky.

Funkce

Vícenásobné návraty, pojmenované návraty, variadické argumenty, defer, closures. Funkce jsou hodnoty.

Základy a vícenásobné návraty
func add(a, b int) int { return a + b }

// idiom Go: (hodnota, error)
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("divide by zero")
    }
    return a / b, nil
}

q, err := divide(10, 3)
Pojmenované návraty
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return      // „naked return“
}
Variadické
func sum(nums ...int) int {
    total := 0
    for _, n := range nums { total += n }
    return total
}

sum(1, 2, 3)
xs := []int{1, 2, 3}
sum(xs...)         // rozbalení
defer
Spouští se při návratu funkce. Pořadí LIFO. Argumenty se vyhodnocují v okamžiku registrace defer.
func readFile(path string) (string, error) {
    f, err := os.Open(path)
    if err != nil { return "", err }
    defer f.Close()    // vždy se spustí při odchodu

    b, err := io.ReadAll(f)
    return string(b), err
}
Hodnoty funkcí a closures
func counter() func() int {
    n := 0
    return func() int {
        n++
        return n
    }
}

next := counter()
fmt.Println(next(), next(), next())  // 1 2 3

Ukazatele

Go má ukazatele, ale nemá ukazatelovou aritmetiku. Existují ze dvou důvodů: vyhnout se kopiím a umožnit volané funkci měnit hodnotu volajícího.

Vzít adresu, dereferencovat
x := 42
p := &x         // ukazatel na x
fmt.Println(*p)  // 42
*p = 100
fmt.Println(x)   // 100
Předání hodnotou vs ukazatelem
func grow(n int)    { n *= 2 }      // bez efektu
func growP(n *int) { *n *= 2 }    // mění volajícího

x := 3
grow(x)        // x je stále 3
growP(&x)      // x je teď 6
new vs &T{}
// new(T) → *T ukazující na nulové T
p := new(int)        // *int ukazující na 0
*p = 7

// kompozitní literál — idiomatičtější
u := &User{Name: "jane"}
Kdy použít ukazatele
Použijte ukazatel: mutace, velká struktura (vyhnout se kopii), splnění rozhraní s ukazatelovými přijímači, volitelné/nil sentinel.
Vyhněte se: malé neměnné struktury, řezy/mapy/kanály (už referenční typy).
Past: braní adresy proměnné smyčky ve starém Go vracelo stejnou adresu v každé iteraci. V Go 1.22+ má každá iterace čerstvou proměnnou, takže &v v range smyčce je bezpečné. Ve starších verzích: dejte si v := v.

Struktury a metody

Struktury drží data. Metody jsou funkce s přijímačem. Vnořování dává kompozici.

Definice a instance
type User struct {
    ID    int      `json:"id"`
    Name  string   `json:"name"`
    Email string   `json:"email,omitempty"`
}

u := User{ID: 1, Name: "jane"}
v := User{1, "jane", ""}   // poziční, křehké
p := &User{Name: "alex"}     // ukazatel na čerstvou strukturu
Metody (hodnotový vs ukazatelový přijímač)
type Counter struct{ n int }

// hodnotový přijímač — pracuje na kopii
func (c Counter) Get() int   { return c.n }

// ukazatelový přijímač — mění volajícího
func (c *Counter) Inc()      { c.n++ }

c := Counter{}
c.Inc()                    // auto &c
fmt.Println(c.Get())
Vnořování (kompozice)
type Logger struct{ Prefix string }
func (l Logger) Log(msg string) {
    fmt.Println(l.Prefix, msg)
}

type Server struct {
    Logger              // vnořeno
    Addr string
}

s := Server{Logger: Logger{"[srv]"}}
s.Log("started")        // povýšená metoda
Anonymní struktura
resp := struct {
    OK   bool
    Data []int
}{OK: true, Data: []int{1, 2, 3}}
Pravidlo: pokud kterákoli metoda potřebuje ukazatelový přijímač, dejte je všem ukazatelové přijímače pro konzistenci. Sada metod hodnotového typu nezahrnuje metody s ukazatelovým přijímačem, což může rozbít plnění rozhraní.

Rozhraní

Rozhraní je sada signatur metod. Cokoli, co všechny implementuje, ho splňuje — implicitně. Žádné klíčové slovo implements.

Definice a splnění
type Stringer interface {
    String() string
}

type Point struct{ X, Y int }

func (p Point) String() string {
    return fmt.Sprintf("(%d,%d)", p.X, p.Y)
}

// Point implicitně splňuje Stringer
var s Stringer = Point{3, 4}
prázdné rozhraní / any
// any === interface{} (Go 1.18+)
var v any = 42
v = "hello"
v = []int{1, 2}

// typové ověření — vytáhne konkrétní typ
n, ok := v.(int)
if ok { fmt.Println("got int", n) }
Běžná stdlib rozhraní
type error interface      { Error() string }
type Stringer interface   { String() string }
type Reader interface     { Read(p []byte) (n int, err error) }
type Writer interface     { Write(p []byte) (n int, err error) }
type Closer interface     { Close() error }
Vnořování rozhraní
type ReadCloser interface {
    Reader
    Closer
}
nil rozhraní ≠ nil ukazatel: proměnná rozhraní drží (typ, hodnotu). Když uložíte nil *MyError do proměnné error, samotné rozhraní není nil. if err != nil bude true, i když podkladový ukazatel je nil. Vyhněte se vracení typovaných nilů jako chyb.

// předem alokovat na známou velikost — vyhne se realokacím

Konkurence

Nesdílejte paměť komunikací; sdílejte paměť komunikováním. Gorutiny + kanály jsou celá filozofie Go.

gorutina
go work(item)             // běží v nové gorutině
go func() {
    fmt.Println("hi")
}()

// gorutiny jsou levné (~2 KB zásobníku)
// runtime je multiplexuje na OS vlákna
kanály
ch := make(chan int)         // nebufferovaný
ch := make(chan int, 10)     // bufferovaný, cap 10

ch <- 42                     // odeslání (blokuje, pokud je plný / bez příjemce)
v := <-ch                    // příjem
v, ok := <-ch                // ok=false na uzavřeném prázdném
close(ch)                    // jen odesílatelé — nikdy příjemce

// range dokud není zavřený
for v := range ch { fmt.Println(v) }
select
select {
case v := <-a:
    fmt.Println("a", v)
case v := <-b:
    fmt.Println("b", v)
case out <- 7:
    fmt.Println("sent")
case <-time.After(time.Second):
    fmt.Println("timeout")
default:
    fmt.Println("nothing ready")
}
sync.WaitGroup
var wg sync.WaitGroup
for _, item := range items {
    wg.Add(1)
    go func(it Item) {
        defer wg.Done()
        work(it)
    }(item)
}
wg.Wait()
sync.Mutex
type Counter struct {
    mu sync.Mutex
    n  int
}
func (c *Counter) Inc() {
    c.mu.Lock()
    defer c.mu.Unlock()
    c.n++
}
context pro zrušení
ctx, cancel := context.WithTimeout(parent, 5*time.Second)
defer cancel()

select {
case <-ctx.Done():
    return ctx.Err()
case v := <-resultCh:
    return v, nil
}

Chyby a generika

Chyby jsou hodnoty. Obalujte je, zkoumejte je, ale nepanikařte při běžném toku.

Vytvořit a obalit
errors.New("bad input")
fmt.Errorf("open %s: %w", path, err)   // %w obaluje

// sentinel
var ErrNotFound = errors.New("not found")

// vlastní typ
type ParseError struct{ Line int; Msg string }
func (e *ParseError) Error() string {
    return fmt.Sprintf("line %d: %s", e.Line, e.Msg)
}
Zkoumat: errors.Is / errors.As
if errors.Is(err, ErrNotFound) {
    // shoda se sentinelem skrz obaly
}

var pe *ParseError
if errors.As(err, &pe) {
    fmt.Println("line", pe.Line)
}

// errors.Join (Go 1.20+)
err := errors.Join(err1, err2)
panic / recover
Použít jen pro skutečně neopravitelné chyby (nebo HTTP middleware na nejvyšší úrovni).
defer func() {
    if r := recover(); r != nil {
        log.Println("recovered:", r)
    }
}()

net/http server

Hello server
package main

import (
    "fmt"
    "net/http"
)

func main() {
    mux := http.NewServeMux()
    mux.HandleFunc("GET /hello/{name}",
        func(w http.ResponseWriter, r *http.Request) {
            name := r.PathValue("name")
            fmt.Fprintf(w, "hi %s", name)
        })
    http.ListenAndServe(":8080", mux)
}

func withLog(h http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        start := time.Now()
        h.ServeHTTP(w, r)
        slog.Info("req", "path", r.URL.Path,
                 "dur", time.Since(start))
    })
}

Tabulkový test
func TestAdd(t *testing.T) {
    for _, tc := range []struct {
        name    string
        a, b, w int
    }{
        {"zero", 0, 0, 0},
        {"pos",  2, 3, 5},
        {"neg", -1, 1, 0},
    } {
        t.Run(tc.name, func(t *testing.T) {
            if g := add(tc.a, tc.b); g != tc.w {
                t.Errorf("got %d, want %d", g, tc.w)
            }
        })
    }
}

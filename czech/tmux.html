1. co je tmux?

tmux je terminálový multiplexer: jedno okno terminálu obsahuje mnoho nezávislých shellů, vedle sebe, v tabech a (nejdůležitější) odpojené od vašeho SSH spojení.

Tři superschopnosti:

  • perzistence — zavřete laptop, zítra se připojte, najdete své shelly přesně tam, kde jste je opustili
  • rozložení — rozdělíte okno na panely; běží vim, tail -f i htop zároveň
  • skriptovatelný — spustíte rozložení projektu jedním příkazem, sdílíte sezení s kolegou, automatizujete setup

2. mentální model

tmux je client/server program. Váš terminál mluví s dlouho běžícím tmux serverem. Server vlastní:

server
└── session "work"
    ├── window 0: editor
    │   ├── pane 0: vim
    │   └── pane 1: shell
    └── window 1: logs
        └── pane 0: tail -f app.log

attach = terminál se připojí k sezení. detach = terminál se odpojí, ale sezení běží dál na serveru. Zavření SSH spojení ≠ zabití sezení.

3. prefix klávesa

Téměř každý příkaz tmuxu uvnitř sezení začíná prefix klávesou. Výchozí je Ctrl+b. Stisknete prefix, uvolníte ho a pak stisknete další klávesu.

Notace: C-b " znamená Ctrl+b a pak " (což je Shift+'). Mnoho lidí přemapuje prefix na Ctrl+a, protože je dosažitelnější.

Vyzkoušejte simulátor níže — stiskněte Ctrl+b a sledujte, jak indikátor začne hořet.

6. panely

Panely rozdělí okno na obdélníky. Notace tmuxu je proslule matoucí:

  • C-b " rozdělí horizontálně (nahoře/dole — horizontální dělič). Mnemo: " jsou dvě tečky nad sebou.
  • C-b % rozdělí vertikálně (vlevo/vpravo — vertikální dělič). Mnemo: % vypadá jako lomítko mezi dvěma boxy.

8. konfigurace — ~/.tmux.conf

Konfiguraci znovu načtete uvnitř tmuxu pomocí C-b : a pak source-file ~/.tmux.conf, nebo si to nabindujte.

# přemapuje prefix na Ctrl-a
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# indexovat okna/panely od 1 (snáze dosáhnete kláves 1..9)
set -g base-index 1
setw -g pane-base-index 1
set -g renumber-windows on

# navigace mezi panely ve stylu vim
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# rozumné rozdělení, které zachovávají aktuální cestu
bind | split-window -h -c "#{pane_current_path}"
bind - split-window -v -c "#{pane_current_path}"

# myš + vi copy mode
set -g mouse on
setw -g mode-keys vi
bind r source-file ~/.tmux.conf \; display "config znovu načten"

# větší scrollback
set -g history-limit 50000

9. pluginy (TPM)

TPM = Tmux Plugin Manager. Přidejte do ~/.tmux.conf:

set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'tmux-plugins/tmux-resurrect'   # uložit/obnovit sezení
set -g @plugin 'tmux-plugins/tmux-continuum'   # auto-obnova při startu
run '~/.tmux/plugins/tpm/tpm'

Uvnitř tmuxu: C-b I instaluje, C-b U aktualizuje. tmux-resurrect umí udělat checkpoint celého stromu sezení; tmux-continuum auto-ukládá každých 15 min a načte při startu — restart laptopu = žádná ztráta práce.

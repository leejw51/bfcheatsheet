환영합니다 — PyTorch에서 첫 신경망

이 페이지를 끝마치면 완전한 신경망 코드를 직접 타이핑해서 정의하고, 학습시키고, 추론까지 실행하게 됩니다. 복사-붙여넣기는 없습니다 — 모든 키 입력이 의미를 가집니다. 각 빈칸에 올바른 토큰을 입력하면 페이지가 점수를 매겨주고, 연속 정답 카운터, 애니메이션 다이어그램, 그리고 브라우저에서 "학습"이 일어나는 미니 프로젝트가 펼쳐집니다.

텐서란 무엇인가?
텐서는 다차원 배열입니다 — 숫자, 벡터, 행렬, 또는 그 이상의 차원을 가진 자료 구조입니다. 신경망 안의 모든 것이 텐서입니다: 입력, 가중치, 그래디언트, 출력 모두요. PyTorch의 torch.Tensor는 NumPy의 ndarray와 거의 똑같지만 GPU에서 실행될 수 있고 자기가 어떻게 만들어졌는지 기억할 수 있습니다(autograd).
모델이란 무엇인가?
학습 가능한 매개변수를 가진 함수입니다. 입력은 층(행렬 곱셈 + 비선형성)을 거쳐 흘러 출력을 만들어냅니다. 출력을 정답과 비교하고, 그래디언트를 계산하여, 가중치를 손실이 줄어드는 방향으로 살짝 밀어 매개변수를 조정합니다.
학습
모델에게 수많은 입력/타깃 쌍을 보여줍니다. 각 쌍마다: 순전파 → 손실 함수 계산 → 역전파(그래디언트) → 옵티마이저 스텝(가중치 갱신). 이를 수천 번 반복합니다. 손실 곡선이 내려갑니다. 그게 전부입니다. 이것이 딥러닝입니다.
추론
학습이 끝나면 가중치를 동결하고 새로운 입력에 대해 순전파만 실행합니다. 더 빠르고(그래디언트 추적이 없음) 결정적입니다. model.eval()과 torch.no_grad()를 사용하세요.

설정

큰 그림 (이것을 외우세요)

for epoch in range(num_epochs):
    for x, y in dataloader:
        pred = model(x)              # 순전파
        loss = loss_fn(pred, y)      # 비교
        optimizer.zero_grad()        # 이전 그래디언트 지우기
        loss.backward()              # 새 그래디언트 계산
        optimizer.step()             # 가중치 갱신

모델 만들기 — 첫 nn.Module

모든 PyTorch 모델은 nn.Module의 서브클래스입니다. __init__에서 층을 정의하고 forward에서 순전파를 정의합니다. 그게 API의 전부입니다.

10클래스 분류용 2층 MLP (전체)
import torch
import torch.nn as nn

class Net(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1  = nn.Linear(784, 128)
        self.relu = nn.ReLU()
        self.fc2  = nn.Linear(128, 10)

    def forward(self, x):
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)
        return x

model = Net()

손실 & 옵티마이저

손실은 "예측이 얼마나 틀렸는지" 알려주는 숫자입니다 — 작을수록 좋습니다. 옵티마이저는 손실이 줄어들도록 그래디언트를 사용해 가중치를 살짝 밀어줍니다. 두 가지 손실 함수가 90% 케이스를, 두 가지 옵티마이저가 95% 케이스를 커버합니다.

손실 함수
회귀 nn.MSELoss — 평균 제곱 오차
분류 nn.CrossEntropyLoss — 소프트맥스 + log + nll 내장
이진 nn.BCEWithLogitsLoss
옵티마이저
단순 optim.SGD(params, lr=0.01)
기본 추천 optim.Adam(params, lr=1e-3)
현대적 optim.AdamW — Adam + 분리된 가중치 감쇠
조합해서 사용하기
import torch.nn as nn
import torch.optim as optim

model     = Net()
loss_fn   = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-3)

16 2단계 — 예측과 타깃 사이의 손실을 계산하세요:

18 4단계 — 역전파로 모든 매개변수에 새 그래디언트 계산:

19 5단계 — 그래디언트를 적용해 가중치를 갱신하세요:

흔한 버그: optimizer.zero_grad()를 잊으면 그래디언트가 배치마다 누적되어 모델이 마치 모든 이전 배치의 합인 것처럼 학습됩니다. 항상 0으로 만드세요.

미니 프로젝트 — XOR 문제로 모델 학습

XOR은 고전적인 "선형 분리 불가능" 문제입니다 — 단일 퍼셉트론은 이를 풀 수 없지만, 2층 MLP는 풀 수 있습니다. 마지막 빈칸을 채워 프로그램을 완성하세요. 모두 초록불이 되면 페이지가 학습을 시뮬레이션해서 손실 곡선이 내려가고 모델이 결정 경계를 실시간으로 학습하는 모습을 보여줍니다.

프로그램을 완성하세요

8개의 빈칸이 남았습니다. 모두 초록색이 되면 모델 학습을 누르세요.

class XORNet(nn.):

    def forward(self, x):
        x = torch.relu(self.fc1(x))

bfgpt — 한 파일에 담긴 작은 GPT

bfgpt는 오후 한 나절에 읽고 노트북으로 학습시킬 수 있는 아기 GPT입니다. 모델, 데이터 준비, 학습, 샘플링을 모두 담은 ~800줄짜리 Python 파일 하나에, 딱 두 개의 의존성 (torch + numpy)만 있으면 됩니다. Karpathy의 nanoGPT를 충실하게 압축한 포트로, GPT-2 다운로드 경로가 제거됐습니다. 토크나이징은 문자 단위로 이루어지므로 토크나이저를 따로 내려받을 필요 없이 텍스트에서 바로 구성됩니다.

self-contained
모델 + 데이터 준비 + 학습 + 샘플링이 모두 bfgpt.py 하나에 담겨 있습니다. 체크포인트에 토크나이저까지 내장돼 있어 샘플링에 다른 것이 필요 없습니다.
two dependencies
torch와 numpy만 있으면 됩니다. transformers도, tiktoken도, 따로 배울 프레임워크도 없습니다.
어디서든 실행
--device auto가 cuda→mps→cpu 순으로 자동 해결합니다. 기본값은 CPU에서도 동작합니다.
문자 토크나이저
sorted(set(text))이 곧 어휘집입니다. tiny-shakespeare는 65개의 고유 문자를 가집니다.
JSONL 입력
한 줄에 {"text": "..."} 하나씩. --text-field로 다른 필드를 지정할 수 있습니다.
~0.8M 파라미터
4 레이어 / 4 헤드 / 폭 128짜리 모델에 block_size=64. 몇 분 안에 셰익스피어풍 텍스트를 생성할 만큼 작습니다.

토크나이저 — 텍스트 자체가 어휘집

다운로드할 토크나이저도, BPE 병합도, 50,257개짜리 토큰 테이블도 없습니다. bfgpt는 코퍼스에서 정렬된 고유 문자 집합을 뽑아 번호를 매깁니다. 그 매핑 (stoi / itos)이 토크나이저 전체이며, meta.pkl에 저장됩니다. 아래에 입력하면 모델이 보는 정확한 encode → ids → decode 왕복 과정을 실시간으로 확인할 수 있습니다.

class CharTokenizer:
    def __init__(self, stoi, itos):
        self.stoi = stoi
        self.itos = itos
        self.vocab_size = len(stoi)

    @classmethod
    def from_text(cls, text):
        chars = sorted(set(text))            # 어휘집
        stoi  = {ch: i for i, ch in enumerate(chars)}
        itos  = {i: ch for i, ch in enumerate(chars)}
        return cls(stoi, itos)

    def encode(self, s):
        return [self.stoi[c] for c in s]   # str -> list[int]

    def decode(self, ids):
        return ''.join(self.itos[int(i)] for i in ids)

  


  

    이 방식의 장점: 데이터의 알파벳이 어휘 크기를 결정하고, id는 2바이트
    uint16에 딱 맞으며 임베딩 테이블이 작습니다. 단점은 시퀀스가 길다는 것
    (문자 하나가 토큰 하나)이므로, 컨텍스트 윈도우 block_size는 단어가 아닌

CausalSelfAttention.forwardbfgpt.py:80

    
def forward(self, x):
    B, T, C = x.size()
    q, k, v = self.c_attn(x).split(self.n_embd, dim=2)   # 행렬 곱 하나, 3방향 분할
    k = k.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)  # (B, nh, T, hs)
    q = q.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)
    v = v.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)

    if self.flash:
        y = F.scaled_dot_product_attention(q, k, v, is_causal=True)
    else:
        att = (q @ k.transpose(-2, -1)) * (1.0 / math.sqrt(k.size(-1)))
        att = att.masked_fill(self.bias[:, :, :T, :T] == 0, float('-inf'))
        att = F.softmax(att, dim=-1)
        y = att @ v
    y = y.transpose(1, 2).contiguous().view(B, T, C)
    return self.resid_dropout(self.c_proj(y))

GPT.forwardbfgpt.py:183

    
def forward(self, idx, targets=None):
    b, t = idx.size()
    pos = torch.arange(0, t, device=idx.device)

    tok_emb = self.transformer.wte(idx)   # (b, t, n_embd)
    pos_emb = self.transformer.wpe(pos)   # (t, n_embd)
    x = self.transformer.drop(tok_emb + pos_emb)
    for block in self.transformer.h:
        x = block(x)
    x = self.transformer.ln_f(x)

    if targets is not None:                 # --- 학습 ---
        logits = self.lm_head(x)
        loss = F.cross_entropy(logits.view(-1, logits.size(-1)),
                               targets.view(-1), ignore_index=-1)
    else:                                  # --- 추론 ---
        logits = self.lm_head(x[:, [-1], :])  # 마지막 위치만
        loss = None
    return logits, loss

  

  
동일한 forward의 두 얼굴: targets가 있으면 모든 위치에 대한 손실을 계산하고 (병렬 학습), 없으면 마지막 위치의 로짓만 처리합니다 — 샘플링에 필요한 것은 그것뿐입니다.

get_lrbfgpt.py:516

      
def get_lr(it):
    if it < warmup_iters:           # 선형 워밍업
        return lr * (it+1) / (warmup_iters+1)
    if it > lr_decay_iters:         # 최솟값
        return min_lr
    ratio = (it - warmup_iters) / (lr_decay_iters - warmup_iters)
    coeff = 0.5 * (1.0 + math.cos(math.pi * ratio))  # 코사인
    return min_lr + coeff * (lr - min_lr)

    

    

      
100회 반복 동안 선형으로 올라가
      learning_rate = 1e-3에 도달한 뒤, 2,000회까지
      코사인 커브를 따라 min_lr = 1e-4로 내려갑니다.

@torch.no_grad()
def generate(self, idx, max_new_tokens, temperature=1.0, top_k=None):
    for _ in range(max_new_tokens):
        idx_cond = idx if idx.size(1) <= self.config.block_size \
                   else idx[:, -self.config.block_size:]   # 컨텍스트로 자르기
        logits, _ = self(idx_cond)
        logits = logits[:, -1, :] / temperature           # 1. 분포 조정
        if top_k is not None:
            v, _ = torch.topk(logits, min(top_k, logits.size(-1)))
            logits[logits < v[:, [-1]]] = -float('Inf')   # 2. top-k 유지
        probs = F.softmax(logits, dim=-1)               # 3. 확률로 변환
        idx_next = torch.multinomial(probs, num_samples=1)  # 4. 하나 뽑기
        idx = torch.cat((idx, idx_next), dim=1)           # 5. 붙이고 반복
    return idx

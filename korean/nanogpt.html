nanoGPT — Karpathy의 PyTorch 약 300줄짜리 미니멀 GPT

karpathy/nanoGPT는 GPT-2 수준을 학습·재현할 수 있는 가장 간단한 엔드투엔드 레포입니다. 거의 모든 일을 두 파일이 처리합니다. model.py가 네트워크를 정의하고, train.py가 학습시킵니다. 이 페이지에서 직접 타이핑해 보면 GPT가 머리에 박힙니다.

왜 nanoGPT인가
한 시간이면 다 읽을 만큼 작지만, OpenWebText에서 GPT-2(124M)를 원본과 같은 손실까지 학습시킬 만큼 큽니다. 단일 PyTorch 파일, 추상화 제로. 모두가 이걸로 배우는 표준 구현체입니다.
무엇을 만드는가
디코더 전용 트랜스포머: 토큰 + 위치 임베딩 → N번 반복되는 (LN → 코잘 어텐션 → 잔차 → LN → MLP → 잔차) → 마지막 LN → 선형 헤드. 이게 GPT입니다.
tiny shakespeare
노트북 GPU에서 몇 분이면 학습되는 약 1MB짜리 셰익스피어 텍스트. 곧바로 재미있는 문자(char) 단위, 또는 제대로 토크나이즈하는 BPE(tiktoken). 둘 다 data/<name>/prepare.py로 제공됩니다.
원하는 만큼 스케일업
같은 코드로 GPT-2를 재현합니다. 노브는 여섯 개: n_layer, n_head, n_embd, block_size, batch_size, lr. torchrun 멀티 GPU는 플래그 하나면 됩니다.

The Block — 8줄짜리 pre-norm 트랜스포머

Block은 두 개의 서브 레이어를 잔차 연결로 감싸고, 각각 LayerNorm을 앞에 둡니다. nanoGPT는 pre-norm을 씁니다 — LN이 잔차 분기 안쪽에 있고, 메인 도로(highway) 위에는 없습니다. 이게 깊은 스택을 학습 가능하게 만드는 트릭입니다.

model.py — Block
class Block(nn.Module):
    def __init__(self, config):
        super().__init__()
        self.ln_1 = LayerNorm(config.n_embd, bias=config.bias)
        self.attn = CausalSelfAttention(config)
        self.ln_2 = LayerNorm(config.n_embd, bias=config.bias)
        self.mlp  = MLP(config)

    def forward(self, x):
        x = x + self.attn(self.ln_1(x))   # 잔차 + pre-norm attn
        x = x + self.mlp(self.ln_2(x))    # 잔차 + pre-norm mlp
        return x

MLP

model.py — MLP (4× 히든, GELU)
class MLP(nn.Module):
    def __init__(self, config):
        super().__init__()
        self.c_fc    = nn.Linear(config.n_embd, 4 * config.n_embd, bias=config.bias)
        self.gelu    = nn.GELU()
        self.c_proj  = nn.Linear(4 * config.n_embd, config.n_embd, bias=config.bias)
        self.dropout = nn.Dropout(config.dropout)

    def forward(self, x):
        return self.dropout(self.c_proj(self.gelu(self.c_fc(x))))

코잘 셀프 어텐션 — GPT의 심장

각 토큰은 학습된 유사도 점수에 따라 가중치를 매겨, 이전 토큰들(과 자기 자신)에서 정보를 섞어 옵니다. 코잘 마스크는 미래 위치를 가려서, 시점 t의 토큰이 t+1을 절대 보지 못하게 합니다. 멀티 헤드는 채널 차원을 n_head개의 독립 어텐션 계산으로 쪼갭니다.

어텐션 히트맵 (T×T)

밝을수록 어텐션 가중치가 큽니다. 온도 슬라이더로 소프트맥스를 날카롭게/부드럽게 조절하세요. 코잘 마스크를 끄면 양방향(BERT 스타일) 어텐션이 됩니다.

model.py — CausalSelfAttention (forward)
def forward(self, x):
    B, T, C = x.size()                            # 배치, 시간, 채널
    q, k, v = self.c_attn(x).split(self.n_embd, dim=2)

    # 헤드로 분할: (B, T, C) → (B, nh, T, hs)  단 C = nh * hs
    q = q.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)
    k = k.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)
    v = v.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)

    att = (q @ k.transpose(-2, -1)) * (1.0 / math.sqrt(k.size(-1)))
    att = att.masked_fill(self.bias[:,:,:T,:T] == 0, float('-inf'))   # 코잘
    att = F.softmax(att, dim=-1)
    att = self.attn_dropout(att)
    y = att @ v                                   # (B, nh, T, hs)
    y = y.transpose(1, 2).contiguous().view(B, T, C)   # 다시 합침
    return self.resid_dropout(self.c_proj(y))

샘플링 — 로짓을 텍스트로 변환

생성은 자기회귀적입니다: 프롬프트를 넣고, 마지막 로짓을 가져와 온도(temperature)를 적용한 뒤, 선택적으로 top-k로 자르고, 소프트맥스, multinomial 샘플, append, 반복.

8개 모의 토큰에 대한 샘플링 분포

슬라이더를 움직여 보세요. τ → 0은 argmax로 날카로워지고, τ → ∞는 균등 분포로 평평해집니다. top_k는 상위 k 외 모두를 0으로 만듭니다.

레시피: logits/τ → 상위 k 밖 마스킹 → softmax → torch.multinomial(probs, 1).
model.py — generate (표준 루프)
@torch.no_grad()
def generate(self, idx, max_new_tokens, temperature=1.0, top_k=None):
    for _ in range(max_new_tokens):
        idx_cond = idx if idx.size(1) <= self.config.block_size else idx[:, -self.config.block_size:]
        logits, _ = self(idx_cond)
        logits = logits[:, -1, :] / temperature        # 마지막 시점만
        if top_k is not None:
            v, _ = torch.topk(logits, min(top_k, logits.size(-1)))
            logits[logits < v[:, [-1]]] = -float('inf')
        probs = F.softmax(logits, dim=-1)
        idx_next = torch.multinomial(probs, num_samples=1)
        idx = torch.cat((idx, idx_next), dim=1)
    return idx

D24 출력 헤드 직전의 마지막 LayerNorm:
x = self.transformer.(x)
logits = self.lm_head(x)
D25 ignore_index = -1로 패딩 라벨 마스킹:
loss = F.cross_entropy(logits.view(-1, logits.size(-1)), targets.view(-1), ignore_index=)
D26 메모리 매핑된 토큰에서 랜덤 배치 읽기.
data = np.memmap('train.bin', dtype=np., mode='r')
ix = torch.randint(len(data) - block_size, (batch_size,))
D27 분산 데이터 병렬용 모델 래핑:
model = torch.nn.parallel.(model, device_ids=[ddp_local_rank])

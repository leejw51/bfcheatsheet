퍼셉트론 — 하나의 뉴런, 두 가지 아이디어

퍼셉트론은 가장 단순한 신경망입니다: 입력의 가중합에 편향을 더하고 부호에 따라 0 또는 1을 발화하는 단일 뉴런이죠. Frank Rosenblatt가 1958년에 도입했고, 오늘날의 모든 딥러닝 신경망은 이 한 가지 아이디어의 변형들을 쌓아 올린 것입니다.

뉴런
입력 $x_1, x_2, \dots, x_n$은 가중치 $w_1, w_2, \dots, w_n$과 곱해져 합산되고, 편향 $b$만큼 이동합니다. 활성화 함수(여기선 step)가 결과를 예측으로 바꿔줍니다.
$\hat{y} = \mathrm{step}(w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b)$
step 활성화
$\mathrm{step}(z) = \begin{cases} 1 & z \ge 0 \\ 0 & z < 0 \end{cases}$

Heaviside 또는 sign 함수라고도 합니다. 뉴런을 "발화"시키는 on/off 스위치죠. 현대 신경망은 이걸 ReLU/sigmoid로 바꿉니다 — step의 도함수가 거의 모든 곳에서 0이라 경사하강법에 부적합하기 때문입니다.

학습
레이블이 붙은 예제를 한 번에 하나씩 보여줍니다. 예측을 계산하고, 틀렸다면 다음번에는 조금 덜 틀리도록 가중치를 살짝 밀어줍니다:
$w_i \leftarrow w_i + \eta\,(y - \hat{y})\,x_i$
$b \leftarrow b + \eta\,(y - \hat{y})$
$\eta$ (eta)는 학습률 (learning rate)입니다. 모든 예제를 올바르게 분류할 때까지 반복합니다.
추론
가중치가 고정되면, 예측은 단 하나의 내적과 step 함수입니다. 그래디언트도 루프도 없이 그냥 $\hat{y} = \mathrm{step}(\mathbf{w}\cdot\mathbf{x} + b)$. 추론 호출은 이게 전부입니다.

def step(z):
    return 1 if z >= 0 else 0

def predict(w, b, x):
    z = sum(wi * xi for wi, xi in zip(w, x)) + b
    return step(z)

수식, 풀어서

2-입력 퍼셉트론의 경우:

$z = w_1 x_1 + w_2 x_2 + b \quad\Longrightarrow\quad \hat{y} = \mathrm{step}(z)$

경계선 $w_1 x_1 + w_2 x_2 + b = 0$이 평면을 두 반평면으로 나눕니다. $z$가 양수면 클래스 1, 음수면 클래스 0입니다.

퍼셉트론 학습 규칙

Rosenblatt의 알고리즘은 즐거울 정도로 단순합니다: 점을 잘못 분류했다면, 가중치 벡터를 그 점 쪽으로 (혹은 반대쪽으로) 살짝 밀어냅니다. 모든 예제에 대해 이걸 반복하면 — 데이터가 선형 분리 가능하다면 — 수렴합니다.

업데이트 — 한 예제 $(x, y)$에 대해
$\hat{y} = \mathrm{step}(\mathbf{w}\cdot\mathbf{x} + b)$
$\mathbf{w} \leftarrow \mathbf{w} + \eta\,(y - \hat{y})\,\mathbf{x}$
$b \leftarrow b + \eta\,(y - \hat{y})$

세 가지 경우:

  • 정답$y - \hat{y} = 0$ — 업데이트 없음.
  • 1을 놓침$y - \hat{y} = +1$ — $\mathbf{w}$를 $\mathbf{x}$ 쪽으로 밂.
  • 오경보$y - \hat{y} = -1$ — $\mathbf{w}$를 $\mathbf{x}$ 반대쪽으로 밂.

흐름 — 다섯 단계

1예제 선택

데이터셋에서 레이블이 붙은 점 $(x, y)$ 하나를 가져옵니다.

2예측

z = w·x + b

3오차 계산

err = y − ŷ
(−1, 0, +1 중 하나)

4가중치 업데이트

w += η · err · x
b += η · err

5반복

오차가 없어질 때까지 모든 예제에 대해 여러 에폭 반복합니다.

참조 구현

퍼셉트론 학습 — 순수 NumPy
import numpy as np

def step(z):
    return np.where(z >= 0, 1, 0)

def train_perceptron(X, y, lr=0.1, epochs=20):
    n, d = X.shape
    w = np.zeros(d)
    b = 0.0
    for epoch in range(epochs):
        for i in range(n):
            z    = np.dot(w, X[i]) + b
            yhat = step(z)
            err  = y[i] - yhat
            w   += lr * err * X[i]   # 가중치 업데이트
            b   += lr * err          # 편향 업데이트
    return w, b

# AND 데이터셋
X = np.array([[0,0],[0,1],[1,0],[1,1]])
y = np.array([ 0,         0,         0,         1      ])

w, b = train_perceptron(X, y)
print(w, b)   # 대략 [0.2, 0.1] -0.2 같은 값
수렴: 퍼셉트론 학습 규칙은 유한한 단계 안에 분리 초평면을 찾는 것이 보장됩니다 — 단, 그런 초평면이 존재할 때만요. 클래스가 겹치거나 XOR 모양이면 영원히 진동합니다. 그 이유는 탭 6에서 봅니다.

추론 — 학습된 퍼셉트론 사용하기

가중치가 고정되면 예측은 한 줄의 수학입니다: 내적, 편향 더하기, step. 루프도, 그래디언트도, 상태도 없습니다. 존재하는 가장 단순한 "신경망 추론"입니다.

스칼라 형식
$\hat{y} = \mathrm{step}\!\left(\sum_{i=1}^{n} w_i x_i + b\right)$
벡터 형식
$\hat{y} = \mathrm{step}(\mathbf{w}^{\!\top}\mathbf{x} + b)$
배치 전체에 대한 벡터화 추론
import numpy as np

# 학습된 가중치 (예: AND용)
w = np.array([1.0, 1.0])
b = -1.5

# 입력 배치, 모양 (N, 2)
X_new = np.array([[0,0], [0,1], [1,0], [1,1]])

logits = X_new @ w + b           # 모양 (N,)
preds  = np.where(logits >= 0, 1, 0)

print(preds)                     # [0 0 0 1]
왜 이렇게 빠를까: 추론은 레이어당 한 번의 행렬-벡터 곱입니다. 진짜 LLM은 수십억 개의 퍼셉트론을 쌓아 올린 것에 불과합니다 — 하지만 각 개별 뉴런의 추론은 이것과 똑같습니다.

XOR 문제 — 뉴런 하나로는 부족한 이유

단일 퍼셉트론은 직선 하나만 그릴 수 있습니다. AND와 OR는 선형 분리 가능합니다 — 직선 하나로 1과 0을 가를 수 있죠. XOR는 그렇지 않습니다. $(0,1)$과 $(1,0)$을 $(0,0)$과 $(1,1)$로부터 분리하는 직선을 그려보세요. 불가능합니다.

마이너스 두 개, 플러스 두 개, 어떤 직선도 갈라놓지 못합니다.

해결책: 퍼셉트론 쌓기 → MLP

퍼셉트론으로 된 은닉층 (hidden layer) 하나를 추가하면 어떤 불리언 함수든 만들 수 있습니다. XOR = (A OR B) AND NOT (A AND B), 두 뉴런이 세 번째 뉴런으로 흘러갑니다.

역사적 배경: Minsky & Papert의 1969년 책 Perceptrons는 XOR가 단일 층으로는 풀 수 없다는 걸 보였습니다. 그 후 "AI 겨울"이 왔습니다. 해결책 — 은닉층을 통한 역전파 (backpropagation) — 는 1980년대까지 널리 채택되지 않았습니다. 그 다층 퍼셉트론(MLP)이 현대 모든 딥러닝 신경망의 직계 조상입니다.

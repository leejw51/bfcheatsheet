변분 오토인코더 — 새는 병목으로 하는 생성 모델링

VAE는 잠재 코드가 단일 점이 아니라 확률 분포인 오토인코더입니다. 인코더는 $(\mu, \log\sigma^2)$를 출력하고, 재매개변수화 트릭으로 $z \sim \mathcal{N}(\mu, \sigma^2)$를 샘플링한 뒤, 디코더가 $\hat{x}$를 재구성합니다. 재구성 손실 + 단위 가우시안에 대한 KL로 학습합니다. 학습이 끝나면 인코더는 버리고 $z \sim \mathcal{N}(0, I)$를 뽑아 새 데이터를 생성합니다.

vs. 오토인코더
평범한 AE는 $x \to z \to \hat{x}$를 결정론적 함수로 압축합니다. 잠재 공간에 구멍이 있어서 — 무작위 $z$를 뽑으면 쓰레기가 나옵니다. VAE는 잠재가 $\mathcal{N}(0, I)$처럼 보이도록 강제해서, 모든 $z$가 그럴듯한 무언가로 디코딩됩니다.
왜 확률적이어야 하나?
우리는 생성을 원합니다: 새 샘플을 그려낼 방법이요. VAE는 $p_\theta(x) = \int p_\theta(x|z)\,p(z)\,dz$를 학습하며, 여기서 $p(z) = \mathcal{N}(0, I)$입니다. $\log p(x)$의 하한 — ELBO — 을 최대화하죠.
그 트릭
샘플링은 미분 불가능합니다. 재매개변수화하세요: $z = \mu + \sigma \odot \varepsilon$, $\varepsilon \sim \mathcal{N}(0, I)$. 이제 $z$가 $\mu$와 $\sigma$만의 함수인 것처럼 그래디언트가 흐릅니다.

한 줄로 보는 ELBO

$\mathcal{L}_{\text{VAE}} = \underbrace{\mathbb{E}_{q(z|x)}[-\log p(x|z)]}_{\text{재구성}} \; + \; \beta \cdot \underbrace{\mathrm{KL}(q(z|x)\,\|\,p(z))}_{\text{정규화}}$

첫 항: 디코더가 샘플링된 $z$로부터 $x$를 얼마나 잘 재구성하는지. 둘째 항: 인코더의 $q(z|x) = \mathcal{N}(\mu, \sigma^2)$가 사전분포 $p(z) = \mathcal{N}(0, I)$로부터 얼마나 멀어졌는지. 인코더는 인코딩을 하면서 사전분포에 가깝게 머물러야 합니다.

이 페이지 사용법: 각 섹션을 읽고 빈칸에 타이핑하세요. Tab으로 다음 빈칸으로 이동, Enter로 확인. Hint 버튼은 모든 빈칸을 보여주고, 다시 클릭하면 숨깁니다.

멘탈 모델

# VAE의 순전파
mu, logvar = encoder(x)                 # 두 헤드
std        = (0.5 * logvar).exp()       # σ = exp(½·logσ²)
eps        = torch.randn_like(std)      # ε ∼ N(0, I)
z          = mu + std * eps             # 재매개변수화
x_hat      = decoder(z)
loss       = recon(x_hat, x) + beta * kl(mu, logvar)

인코더는 두 개의 벡터를 만듭니다

하나의 잠재 벡터를 만드는 일반 AE와 달리, VAE 인코더에는 두 개의 출력 헤드가 있습니다: 하나는 평균 $\mu$용, 다른 하나는 $\log \sigma^2$ (로그 분산) 용 — $\sigma$ 대신 로그 분산을 예측하면 네트워크 출력을 제약 없이 둘 수 있고, $\exp$ 후에는 $\sigma$가 양수로 유지됩니다.

PyTorch로 구현한 인코더
import torch
import torch.nn as nn

class Encoder(nn.Module):
    def __init__(self, in_dim=784, hidden=400, latent=20):
        super().__init__()
        self.fc1     = nn.Linear(in_dim, hidden)
        self.fc_mu   = nn.Linear(hidden, latent)   # 평균 헤드
        self.fc_lv   = nn.Linear(hidden, latent)   # 로그 분산 헤드

    def forward(self, x):
        h = torch.relu(self.fc1(x))
        return self.fc_mu(h), self.fc_lv(h)        # (μ, log σ²)
디코더 — 역방향 여행
class Decoder(nn.Module):
    def __init__(self, latent=20, hidden=400, out_dim=784):
        super().__init__()
        self.fc1 = nn.Linear(latent, hidden)
        self.fc2 = nn.Linear(hidden, out_dim)

    def forward(self, z):
        h = torch.relu(self.fc1(z))
        return torch.sigmoid(self.fc2(h))    # 픽셀은 [0, 1]

재매개변수화 — 모든 걸 미분 가능하게 만드는 트릭

샘플 $z \sim \mathcal{N}(\mu, \sigma^2)$를 통해 역전파해야 합니다. 순수 무작위 샘플링은 그래디언트가 없습니다 — "분포에서 뽑기"를 통과하는 연쇄 법칙 경로가 없거든요. 해결책: 무작위성을 바깥으로 빼냅니다.

손실 — 재구성 + KL = ELBO

ELBO를 최대화하는 것은 두 항을 최소화하는 것과 같습니다: 디코더가 $x$를 얼마나 못 재구성하는지, 그리고 인코더의 분포가 $\mathcal{N}(0, I)$로부터 얼마나 멀어졌는지.

$\mathcal{L} = \underbrace{\mathrm{BCE}(\hat{x}, x)}_{\text{재구성}} \; + \; \beta \cdot \underbrace{-\tfrac{1}{2}\sum_{j}\bigl(1 + \log\sigma_j^2 - \mu_j^2 - \sigma_j^2\bigr)}_{\mathrm{KL}(\,\mathcal{N}(\mu,\sigma^2)\,\|\,\mathcal{N}(0,I)\,)}$
재구성 항
이진 / 베르누이 픽셀(MNIST)에는 픽셀별 F.binary_cross_entropy, 가우시안 픽셀에는 F.mse_loss를 씁니다. 분석적 ELBO 유도와 맞추려면 reduction='sum'을 쓴 다음 배치 크기로 나눠주세요.
KL 항 — 닫힌 형태
두 가우시안의 KL은 깔끔한 분석적 형태입니다. 샘플링 불필요: 
kl = -0.5 * torch.sum(
    1 + logvar - mu.pow(2) - logvar.exp()
)

코드 드릴 — 전체 VAE 조립하기

이제 모든 걸 묶어주는 부분을 타이핑하세요: reparameterize, 전체 순전파, 손실 함수, 그리고 한 번의 학습 스텝.

정석적인 VAE 순전파 + 손실 (전체)
import torch
import torch.nn as nn
import torch.nn.functional as F

class VAE(nn.Module):
    def __init__(self, in_dim=784, hidden=400, latent=20):
        super().__init__()
        self.fc1     = nn.Linear(in_dim, hidden)
        self.fc_mu   = nn.Linear(hidden, latent)
        self.fc_lv   = nn.Linear(hidden, latent)
        self.fc3     = nn.Linear(latent, hidden)
        self.fc4     = nn.Linear(hidden, in_dim)

    def encode(self, x):
        h = torch.relu(self.fc1(x))
        return self.fc_mu(h), self.fc_lv(h)

    def reparameterize(self, mu, logvar):
        std = (0.5 * logvar).exp()
        eps = torch.randn_like(std)
        return mu + std * eps

    def decode(self, z):
        h = torch.relu(self.fc3(z))
        return torch.sigmoid(self.fc4(h))

    def forward(self, x):
        mu, logvar = self.encode(x.view(-1, 784))
        z          = self.reparameterize(mu, logvar)
        return self.decode(z), mu, logvar

def vae_loss(x_hat, x, mu, logvar, beta=1.0):
    recon = F.binary_cross_entropy(x_hat, x.view(-1, 784), reduction='sum')
    kl    = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp())
    return recon + beta * kl

1. what is tmux?

tmux is a terminal multiplexer: one terminal window holds many independent shells, side by side, in tabs, and (most importantly) detached from your SSH connection.

Three superpowers:

  • persistence — close the laptop, reconnect tomorrow, find your shells exactly where you left them
  • layouts — split a window into panes; run vim, tail -f, and htop together
  • scriptable — start a project layout with one command, share a session with a teammate, automate setup

2. mental model

tmux is a client/server program. Your terminal talks to a long-running tmux server. The server owns:

server
└── session "work"
    ├── window 0: editor
    │   ├── pane 0: vim
    │   └── pane 1: shell
    └── window 1: logs
        └── pane 0: tail -f app.log

attach = your terminal connects to a session. detach = your terminal disconnects, but the session keeps running on the server. Closing the SSH connection ≠ killing the session.

3. the prefix key

Almost every tmux command inside a session starts with the prefix key. The default is Ctrl+b. You press the prefix, release it, then press the next key.

Notation: C-b " means Ctrl+b then " (which is Shift+'). Many people remap the prefix to Ctrl+a because it's easier to reach.

Try the simulator below — press Ctrl+b and watch the indicator go hot.

6. panes

Panes split a window into rectangles. tmux's notation is famously confusing:

  • C-b " splits horizontally (top/bottom — a horizontal divider). Mnemonic: " is two stacked dots.
  • C-b % splits vertically (left/right — a vertical divider). Mnemonic: % looks like a slash between two boxes.

7. copy mode & scrollback

Your mouse can scroll the terminal, but in tmux the real way is copy mode. Enter it with C-b [. Now you can move with vi keys, search, and yank text.

8. configuration — ~/.tmux.conf

Reload the config from inside tmux with C-b : then source-file ~/.tmux.conf, or bind it.

# remap prefix to Ctrl-a
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# start window/pane indexing at 1 (easier to reach 1..9 keys)
set -g base-index 1
setw -g pane-base-index 1
set -g renumber-windows on

# vim-style pane navigation
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# sane splits that keep current path
bind | split-window -h -c "#{pane_current_path}"
bind - split-window -v -c "#{pane_current_path}"

# mouse + vi copy mode
set -g mouse on
setw -g mode-keys vi
bind r source-file ~/.tmux.conf \; display "config reloaded"

# bigger scrollback
set -g history-limit 50000

9. plugins (TPM)

TPM = Tmux Plugin Manager. Append to ~/.tmux.conf:

set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'tmux-plugins/tmux-resurrect'   # save/restore sessions
set -g @plugin 'tmux-plugins/tmux-continuum'   # auto-restore on boot
run '~/.tmux/plugins/tpm/tpm'

Inside tmux: C-b I installs, C-b U updates. tmux-resurrect can checkpoint full session trees; tmux-continuum auto-saves every 15 min and reloads on boot — laptop reboot = no work lost.
